
Usage of audio_beep
===========================

Usage
---------------------------

Select options in below.

- [CXD56xx Configuration]
    [Audio] <= Y
- [Examples]
    [Audio beep example] <= Y

Or use audio_beep default configuration

$ ./tools/config.py examples/audio_beep

Build and install
--------------------------

Type 'make' to build SDK.
Install 'nuttx.spk' to system.

Execute
--------------------------

Type 'audio_beep' on nsh.
nsh>audio_beep

Output a beep of the set frequency.
